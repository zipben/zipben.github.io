Card Deck Simulator

Card Deck Simulator is a tool for simulating card draws in tabletop role-playing games. It allows you to create a custom deck, shuffle it, and draw cards to simulate gameplay scenarios.

Use the input fields below to set the count for each of the 5 suits: lore, empathy, harmony, grace, and power. You can also take an injury, add a thread, or add an omen to the deck.
